# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it responsibly by contacting the maintainer at abdulbasit18080@gmail.com.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Security Best Practices

- Do not commit `.env` files or credentials
- Use secure file I/O practices
- Validate all user inputs before processing
